# Cuadro_mando_empresarial_Lab1
Repositorio para el laboratorio #1 que corresponde al cuadro de mando integral con los datos generales de una empresa.

Mediante este laboratorio se busca crear un cuadro de mando para mostrar el comportamiento y el último valor de 
los indicadores: índice de innovación, inversión en capacitación, el porcentaje de 
satisfacción del cliente y el número de transacciones. 

Por lo cual se desarrollaron algunos graficos y se incluyeron filtros para lograr este objetivo.
Los datos fueron brindados por el profesor.
